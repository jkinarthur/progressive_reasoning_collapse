### Compile
```bash

bibtex main

```

Or simply:
```bash
latexmk -pdf main.tex
```

## Figure Generation

The figures should be generated by running the experiments:

```bash
cd ../experiments
python main.py --experiments all --datasets ml-1m amazon-beauty amazon-toys yelp
```

This will generate:
1. `exp2_critical_depth_*.pdf` - Critical depth phase transition diagrams
2. `exp6_layerwise_clusters_*.pdf` - t-SNE/PCA visualizations
3. `exp3_comparison_*.pdf` - Radar charts and comparison plots

Copy the generated figures to `paper/figures/` and update the filenames in `main.tex`.
